# Computer Vision: Recognize Hand Postures with MediaPipe and OpenCV

## Project Overview

Developed a computer vision system using MediaPipe and OpenCV to recognize hand postures in real-time. The system is also capable of performing specific actions, such as opening a picture, based on the detected hand gestures.

- **Hand Posture Recognition**: Implemented a real-time hand tracking and posture recognition system using MediaPipe's hand landmarks detection.

- **Gesture-Based Interaction**: Enabled gesture-based commands, such as opening a picture when a specific hand posture is detected.

- **Seamless Integration**: Combined MediaPipe's robust hand tracking capabilities with OpenCV for image processing and interaction handling.

## Technologies Used

- **MediaPipe**: For real-time hand tracking and extracting hand landmarks.
- **OpenCV**: For image processing and implementing gesture-triggered actions.
- **Python**: As the primary programming language for integrating MediaPipe and OpenCV functionalities.

## Challenges and Learnings

One of the main challenges was ensuring accurate recognition of various hand postures under different lighting conditions and camera angles. This required fine-tuning the detection thresholds and preprocessing the input frames to improve robustness. Additionally, mapping specific gestures to actions, such as opening a picture, required carefully designing gesture-to-action mappings to avoid false triggers.

## Outcome

The system successfully recognizes multiple hand postures in real-time and performs specific actions based on user-defined gestures. This project demonstrates how computer vision can be leveraged for intuitive human-computer interaction, providing a foundation for gesture-based applications in various domains such as gaming, accessibility tools, and smart home systems.

# 電腦視覺：使用 MediaPipe 和 OpenCV 識別手勢姿態

## 項目概述

開發了一個基於 MediaPipe 和 OpenCV 的電腦視覺系統，能夠實時識別手勢姿態。該系統還支持基於手勢觸發特定操作，例如打開圖片。

- **手勢姿態識別**：使用 MediaPipe 的手部關鍵點檢測功能，實現了實時的手勢跟蹤和姿態識別。

- **基於手勢的交互**：支持基於手勢的指令，例如檢測到特定手勢時自動打開圖片。

- **無縫集成**：將 MediaPipe 強大的手部跟蹤功能與 OpenCV 的圖像處理和交互處理能力相結合。

## 使用的技術

- **MediaPipe**：用於實時手部跟蹤並提取手部關鍵點。
- **OpenCV**：用於圖像處理以及實現基於手勢觸發的操作。
- **Python**：作為主要編程語言，用於整合 MediaPipe 和 OpenCV 功能。

## 挑戰與學習

主要挑戰之一是確保在不同光照條件和攝像頭角度下準確識別各種手勢姿態。這需要調整檢測閾值並對輸入幀進行預處理以提高穩定性。此外，將特定手勢映射到操作（如打開圖片）需要精心設計手勢與操作之間的映射，以避免誤觸發。

## 成果

該系統成功實現了多種手勢姿態的實時識別，並能根據用戶定義的手勢執行特定操作，例如打開圖片。這個項目展示了如何利用電腦視覺技術實現直觀的人機交互，為遊戲、輔助工具和智能家居系統等領域的基於手勢應用奠定了基礎。
